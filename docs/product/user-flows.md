# 사용자/운영자 화면 흐름 및 권한

## Member 플로우
1. 홈 진입
2. 구장 목록에서 필터/정렬 적용
3. 구장 상세에서 해당 구장 모임 목록 확인
4. 모임 상세에서 가입 신청
5. 마이페이지에서 신청 상태 및 알림 확인

## Host 플로우
1. 내 모임 진입
2. 가입 신청 목록 확인
3. 승인/거절 처리
4. 정모 일정 생성/수정
5. 일정 변경 시 참여자 알림 발송

## 권한 매트릭스
- Guest: 조회만 가능
- Member: 가입 신청, 신청 취소(승인 전), 알림 조회
- Host: 본인 모임 신청 처리, 일정 관리, 모임 정보 수정
- Admin: 전체 운영 권한

## 예외 처리
- 승인 대기 중 재신청: 차단 + 기존 신청 상태 안내
- 모집 마감: 신청 차단
- 정원 초과: 승인 차단
