# Phase 1~3 우선순위 백로그

## Phase 1 (4~6주)
- 소셜 로그인(카카오/네이버/구글)
- 구장 목록/상세/필터
- 구장별 모임 조회
- 모임 가입 신청/승인
- 정모 일정 및 기본 알림

## Phase 1.5 (MVP 직후)
- 번개 매칭
- 알림 고도화(개인화 리마인드, 중요 공지 우선)
- 즐겨찾기(구장/모임)

## Phase 2 (3~4주)
- 후기/평점
- 채팅 고도화(공지 고정, 투표, 읽음 상태, 모더레이션)
- 추천(레벨/거리/시간대)
- 신고/패널티 기본 정책

## Phase 3 (3~5주)
- 서울/경기권 외 지역 확장
- 결제/정산/환불
- 고급 통계(혼잡도/랭킹)

## 마일스톤
- M1: 구장/모임 읽기 흐름 완성
- M2: 가입 신청/승인 E2E 완성
- M3: 운영자 일정 관리 + 알림 안정화
- M4: MVP 베타 릴리즈 및 지표 수집
